'/vendor/autoload.php';```## Obtain your personal API tokenOpen the [Todoist web app](https://todoist.com), click on the gear icon, select ‚ÄúSettings‚Äù, then ‚ÄúIntegrations‚Äù. Your API token is listed on the bottom of this page.## Usage```php<?php$Todoist = new FabianBeiner\Todoist\Todoist('YOUR_API_TOKEN');```## Methods & Examples### Get all
